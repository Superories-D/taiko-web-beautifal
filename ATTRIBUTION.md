# Attribution

taiko.asia is an unofficial fan-made Taiko Web project.

This project descends from the original open-source Taiko Web project formerly hosted at taiko.bui.mp. After the original site and repository became unavailable, the community continued development through mirrors, forks, and derivative projects.

taiko.asia was later independently maintained and expanded by Anthony, with major changes including official-chart organization, multilingual SEO, rankings, song search improvements, custom chart support, deployment improvements, and UI/UX changes.

This project is not affiliated with Bandai Namco, the original Taiko Web team, taikoapp.uk, or cjdgrevival.
